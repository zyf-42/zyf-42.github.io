hexo个人博客搭建教程

Hexo简介

Hexo是一款基于Node.js的静态博客框架，依赖少，易于安装使用，可以方便的生成静态网页托管在GitHub进行详细查看，因为Hexo的创建者是台湾人，对中文的支持很友好，可以选择中文进行查看。这里使用Gitee和windows。

Hexo搭建步骤

  1. 安装Git
  2. 安装Node.js
  3. 安装Hexo
  4. Gitee创建个人仓库
  5. 生成SSH添加到Gitee
  6. 将hexo部署到Gitee

安装Git

Git是目前世界上最先进的分布式版本控制系统，可以有效、高速的处理从很小到非常大的项目版本管理。也就是用来管理你的hexo博客文章，上传到GitHub的工具。
windows：到git官网上下载,Download git,下载后会有一个 Git Bash 的命令行工具，以后就用这个工具来使用git。

安装nodejs

Hexo是基于nodeJS编写的，所以需要安装一下nodeJs和里面的npm工具。
windows：nodejs自行选择windows的版本。
然后你就可以使用Git Bash来敲命令行了。
